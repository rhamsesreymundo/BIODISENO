PROTOTIPO FINAL

El prototipo tres se siguio la misma forma que el prototipo 2 con algunas modiciaciones. Se modifico el radio de curvatura el cual permitira que el brazo se flexione y no sienta incomodidad. La parte inferior se cerro para que el uñon este protegido. Se elimino los soporte. Los orificios fueron cambiados de posición para permitir que las correas se coloquen y no crear vulto. Se redondio todas las esquinas del protitipo para que no lastime el muñon. Para un mejor ajuste se adiciono un soporte anclado el cual permita que el dispositivo mantenga su posición 1.El radio de la circunferencia es pequeño y provoca molestias al momento de flexionar el brazo.
